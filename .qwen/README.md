# Skills Configuration

## Skills Directory
Skills are linked from: `F:\ffff\agentflow-pro\.claude\skills\`

This includes ~300 skills for:
- API Integrations (Airtable, GitHub, Gmail, Google Calendar, etc.)
- Database (Supabase, Notion, etc.)
- Communication (Slack, Zoom, etc.)
- Content (Contentful, YouTube, etc.)
- And many more...

## MCP Servers
MCP configuration linked from: `F:\ffff\agentflow-pro\.vscode\mcp.json`

This includes ~60 MCP servers for various integrations.

## Usage
When working on this project:
1. Skills are automatically available via the linked directory
2. MCP servers are configured via mcp.json
3. Git and Node.js need to be installed separately

## Installing Dependencies
```bash
npm install
```

## Git Setup
Install Git from: https://git-scm.com/download/win

## Node.js Setup
Install Node.js from: https://nodejs.org/
