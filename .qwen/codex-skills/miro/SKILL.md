### MIRO_CREATE_CONNECTOR

**MCP Tool**: `mcp__composio__MIRO_CREATE_CONNECTOR`

Tool to create a connector (edge/arrow) that links two existing board items. Use after creating or locating the two endpoint items (shapes/cards/stickies) to establish visual relationships in flowcharts and diagrams. Both start and end items must be specified; connectors cannot be created with loose endpoints.

**Parameters:**
  - `shape` (string, required): Shape/style of the connector line.
  - `style` (object, optional): Visual styling properties for the connector line and caption.
  - `endItem` (object, required): Defines the ending point of the connector. Must specify either snapTo or position.
  - `board_id` (string, required): Unique identifier of the board where the connector will be created.
  - `captions` (array, optional): Array of caption objects to display text along the connector.
  - `startItem` (object, required): Defines the starting point of the connector. Must specify either snapTo or position.

### MIRO_CREATE_CONNECTOR2

**MCP Tool**: `mcp__composio__MIRO_CREATE_CONNECTOR2`

Tool to create a connector between two items on a Miro board. Use after creating or identifying two board items to establish a visual connection. The startItem.id must be different from endItem.id.

**Parameters:**
  - `shape` (string, optional): The path type of the connector line, defines curvature. Default: curved.
  - `style` (object, optional): Contains information about the style of a connector, such as the color or caption font size.
  - `endItem` (object, required): The end point of the connector. endItem.id must be different from startItem.id.
  - `board_id` (string, required): Unique identifier (ID) of the board for which you want to create the connector.
  - `captions` (array, optional): Blocks of text you want to display on the connector. Maximum 20 captions.
  - `startItem` (object, required): The start point of the connector. startItem.id must be different from endItem.id.
