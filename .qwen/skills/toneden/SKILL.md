  - `target` (string, optional): URL or object ID to promote.
  - `currency` (string, optional): Currency code for the budget (e.g., USD, EUR).
  - `platform` (string, required): The advertising platform to use (facebook or google).
  - `audiences` (array, required): Array of audience objects for targeting. Structure varies by platform.
  - `creatives` (object, required): Creative assets for the campaign. Structure varies by platform.
  - `objective` (string, optional): Marketing objective for the campaign.
  - `targeting` (object, optional): Additional targeting parameters for the campaign.
  - `budget_type` (string, required): Budget allocation strategy (daily or lifetime).
  - `target_type` (string, optional): Campaign objective type (e.g., conversions, reach, engagement).
  - `budget_amount` (number, required): Numeric budget value in the specified currency.
  - `end_timestamp` (integer, optional): UNIX timestamp for when the campaign should end.
  - `is_continuous` (boolean, optional): If true, the campaign runs indefinitely (no end date).
  - `report_emails` (array, optional): Email addresses to receive campaign reports.
  - `optimization_id` (string, optional): Optimization goal identifier for the campaign.
  - `start_timestamp` (integer, optional): UNIX timestamp for when the campaign should start.
  - `target_link_url` (string, optional): URL to direct users to when they click the ad.
  - `external_poster_id` (string, optional): ID of the external poster/page for the campaign.
  - `instagram_account_id` (string, optional): Instagram account ID for Instagram placements (Facebook platform).
  - `external_ad_account_id` (string, required): ID of the external ad account on the platform.
  - `disable_all_notifications` (boolean, optional): If true, disables all email notifications for this campaign.

### TONEDEN_CREATE_ATTACHMENT

**MCP Tool**: `mcp__composio__TONEDEN_CREATE_ATTACHMENT`

Tool to create an attachment. Use when you need to programmatically create a social unlock or contest attachment after gathering all required parameters.

**Parameters:**
  - `type` (string, required): Attachment type: 'unlock' for social unlock or 'contest' for a contest attachment.
  - `platform` (string, optional): Social action for single-action attachments. Required when type='unlock' and only one social action.
  - `platforms` (array, optional): List of social actions for multi-action attachments. Required when type='contest' or multiple unlock actions.
  - `unlock_text` (string, optional): Coupon code text to display. Required when unlock_type='coupon'.
  - `unlock_type` (string, optional): Reward type for social unlocks. Required when type='unlock'.
  - `download_url` (string, optional): URL of the file to download or link target. Required when unlock_type='download' or 'link'.
  - `messenger_page_id` (string, optional): Facebook Page ID to subscribe on Messenger. Required when platform='facebook-messenger'.
  - `stream_unlock_url` (string, optional): URL of the unlisted video to stream. Required when unlock_type='stream'.

### TONEDEN_GET_LINK_INSIGHTS

**MCP Tool**: `mcp__composio__TONEDEN_GET_LINK_INSIGHTS`

Tool to retrieve detailed insights for a ToneDen link. Use when you have a link ID and need metrics such as clicks, clickthroughs, email clicks, etc.

**Parameters:**
  - `linkID` (string, required): Numeric ID of the link to retrieve performance data for.
  - `metric` (string, required): Which action type to query results for. 'rsvps' is only a valid value for livestream links.
  - `end_date` (string, optional): If provided, only actions that occurred before the specified date will be counted. Should be in ISO 8601 format.
  - `start_date` (string, optional): If provided, only actions that occurred after the specified date will be counted. Should be in ISO 8601 format.
  - `aggregate_geo` (string, optional): Groups results by visitor location.
  - `for_distinct_fans` (boolean, optional): If true, only one action per fan will be counted in the stats.
  - `aggregate_interval` (string, optional): Groups results by specified time interval.
  - `aggregate_by_service` (boolean, optional): Groups results by service clicked.
  - `aggregate_by_referrer` (boolean, optional): Groups results by referring domain.
