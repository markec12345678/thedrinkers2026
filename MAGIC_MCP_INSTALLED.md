## 🚀 KAKO NAMESTITI

### **1. Install Dependencies:**

```bash
cd magic-mcp-21st
npm install
```

### **2. Build:**

```bash
npm run build
```

### **3. Configure MCP Server:**

**Za Cursor:**

```json
// .cursor/mcp.json
{
  "mcpServers": {
    "magic-mcp": {
      "command": "node",
      "args": ["F:/thedrinkers/the/magic-mcp-21st/dist/index.js"],
      "env": {
        "API_KEY": "your-api-key-from-21st.dev"
      }
    }
  }
}
```

### **4. Get API Key:**

```
1. Obišči https://21st.dev
2. Sign up for free account
3. Get API key from dashboard
4. Add to .env ali MCP config
```

### **5. Test:**

```
V Cursor/Windsurf:
/ui create a button with gradient
```
