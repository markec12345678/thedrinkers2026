## 🎯 11. OPTIMIZATION PLAYBOOK

### If CTR < 1%:

```
PROBLEM: Creative ni engaging

SOLUTIONS:
1. Test new headline (more specific)
2. Change image/video (show crowd/band)
3. Add urgency ("Limited tickets")
4. Test different CTA copy
5. Narrow audience targeting
```

### If CPC > 2€:

```
PROBLEM: Bidding too high or relevance low

SOLUTIONS:
1. Improve ad relevance score
2. Narrow audience (more specific)
3. Test different placements (Stories cheaper)
4. Lower bids gradually
5. Focus on organic + retargeting
```

### If Conversion Rate < 2%:

```
PROBLEM: Landing page or offer issue

SOLUTIONS:
1. Simplify checkout process
2. Add trust badges
3. Show social proof (X tickets sold)
4. Add urgency (X people viewing)
5. Test different price points
6. Add payment options
```
