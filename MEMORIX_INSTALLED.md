## 🎯 KAJ JE MEMORIX?

### **Opis:**

```
Memorix = Cross-Agent Memory Bridge

MCP server ki:
- Git Memory (commits as memory)
- Reasoning Memory (decision tracking)
- Observation Memory (changes)
- Cross-agent recall (multi-IDE)
- Local-first (privacy)
```

### **Kaj Doda:**

```
✅ Git Memory - turn commits into searchable memory
✅ Reasoning Memory - store why decisions were made
✅ Observation Memory - store what changed
✅ Cross-Agent Recall - works with Cursor, Windsurf, Claude Code, etc.
✅ Memory Quality Pipeline - formation, compaction, retention
✅ Control Plane Dashboard - web UI for collaboration
```

## 🔧 FUNKCIONALNOSTI (MCP Tools)

### **Core Tools:**

```
1. memorix_store - Store memories
2. memorix_search - Search memories
3. memorix_detail - Get memory details
4. memorix_timeline - Get timeline of memories
5. memorix_resolve - Resolve memory references
6. memorix_git_hook - Install git post-commit hook
7. memorix_ingest_commit - Ingest git commits
8. memorix_session_start - Start new session
9. memorix_session_end - End session
10. memorix_project_init - Initialize project
```

### **Memory Types:**

```
✅ Git Memory - from git commits (source='git')
✅ Reasoning Memory - decision rationale
✅ Observation Memory - what changed
✅ Session Memory - handoff context
```

## 🚀 KONFIGURACIJA

### **MCP Config (.qwen/mcp.json):**

```json
{
  "mcpServers": {
    "memorix": {
      "command": "memorix",
      "args": ["serve"]
    }
  }
}
```

### **Initialize Project:**

```bash
cd F:\thedrinkers\the
memorix init
```

### **Start Memorix:**

```bash
# Normal mode (stdio MCP)
memorix serve

# HTTP mode (with dashboard)
memorix serve-http --port 3211
```

### **Git Integration:**

```bash
# Install post-commit hook
memorix git-hook --force

# Or manual ingest
memorix ingest commit
memorix ingest log --count 20
```
