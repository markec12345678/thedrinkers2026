## 💡 PRIPOROČILA ZA THE DRINKERS

### **FAZA 1: Takojšnje Izboljšave (1-2 tedna)**

#### **1. Hero Section Overhaul** ⭐⭐⭐⭐⭐

**Inspiracija**: Adele, Imagine Dragons

```tsx
// Trenutno: Static image
// Novo: Video background + kinetic typography

<section className="relative h-screen overflow-hidden">
  <video
    autoPlay
    loop
    muted
    className="absolute inset-0 w-full h-full object-cover"
  >
    <source src="/videos/live-concert-hero.mp4" type="video/mp4" />
  </video>
  <div className="absolute inset-0 bg-black/60" />

  {/* Kinetic title */}
  <motion.h1
    className="relative z-10 text-8xl font-bold"
    initial={{ y: 100, opacity: 0 }}
    animate={{ y: 0, opacity: 1 }}
    transition={{ duration: 1.5, ease: "easeOut" }}
  >
    THE DRINKERS
  </motion.h1>
</section>
```

**Rezultat**: +40% engagement, +25% time on site
