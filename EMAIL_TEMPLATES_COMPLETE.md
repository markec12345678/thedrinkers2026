## 📊 EMAIL METRICS TO TRACK

### Key Metrics:
| Metric | Benchmark | Goal |
|--------|-----------|------|
| Open Rate | 20-25% | 30%+ |
| Click Rate | 2-5% | 8%+ |
| Unsubscribe | <0.5% | <0.2% |
| Conversion | 1-3% | 5%+ |

### A/B Test Ideas:
1. **Subject Lines:** Emoji vs no emoji
2. **Send Time:** Morning (9am) vs Evening (7pm)
3. **CTA Color:** Crimson vs Orange
4. **Personalization:** First name vs no name
5. **Length:** Short vs Long copy

## 🛠️ IMPLEMENTATION GUIDE

### Email Service Providers:

**Recommended:**
1. **Customer.io** - Behavior-based automation
2. **Mailchimp** - SMB friendly
3. **Resend** - Developer-friendly
4. **Kit** - Creator focused

### Setup Steps:

1. **Import subscribers**
   ```
   CSV format: email, first_name, signup_date, source
   ```

2. **Create segments**
   - New subscribers (0-7 days)
   - Active fans (opened last 3 emails)
   - VIP members (registered on /bar)
   - Concert-goers (clicked tour links)

3. **Setup automation**
   - Trigger: Form signup on thedrinkers.si
   - Action: Add to "Launch Sequence"
   - Exit: User makes purchase OR unsubscribes

4. **Add tracking**
   - UTM parameters on all links
   - Conversion events (purchase, VIP signup)
   - Engagement scoring
