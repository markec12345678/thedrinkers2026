## 📐 SPECIFIKACIJE PO PLATFORMAH

### 1. META (Facebook/Instagram)

#### Image Ads:

**Feed Post (Square):**
```
Size: 1080x1080px
Format: JPG or PNG
Max File Size: 30MB
Text on Image: < 20% površine

Content:
- Band photo (high quality)
- Concert date & venue
- "Early Bird" badge
- CTA button overlay

Colors:
- Primary: Crimson Red (#dc143c)
- Secondary: Black (#0a0a0a)
- Accent: White/Silver

Text Overlay:
"THE DRINKERS LIVE"
"[DATUM] | [VENUE]"
"VSTOPNICE OD [CENA]€"
```

**Feed Post (Portrait - Better Performance):**
```
Size: 1080x1350px
Format: JPG or PNG
Same content as square, optimized for mobile
```

**Story/Reels:**
```
Size: 1080x1920px (full vertical)
Format: JPG, PNG, or MP4
Duration: Up to 15 seconds (image) or 30s (video)

Content:
- Full-screen band photo or video
- Date & venue (top third)
- "Swipe Up" CTA (bottom third)
- Animated elements OK

Safe Zones:
- Top 250px: No text (UI overlay)
- Bottom 250px: No text (Swipe Up area)
```

#### Video Ads:

**Feed Video:**
```
Size: 1080x1080px or 1080x1350px
Duration: 15-30 seconds
Format: MP4 or MOV
Max File Size: 4GB
Aspect Ratio: 1:1 or 4:5

Structure:
0-3s:  Hook (band logo + text "En Sam Koncert!")
3-8s:  Problem ("Si vedel da igrajo samo enkrat?")
8-20s: Solution (concert footage, crowd going wild)
20-30s: CTA (poster with date, venue, ticket link)
```

**Story/Reels Video:**
```
Size: 1080x1920px
Duration: 15-30 seconds
Format: MP4
Captions: REQUIRED (85% watch without sound)

Structure:
0-3s:  Pattern interrupt (flash lights + band name)
3-10s: Best concert moments (quick cuts)
10-20s: Date, venue, ticket info
20-30s: CTA with urgency ("Early bird ends [date]")
```

## 💡 CREATIVE TIPS & BEST PRACTICES

### Do's:

```
✅ Use real concert footage (authentic > polished)
✅ Show crowd energy (social proof)
✅ Include clear date & venue (information)
✅ Use urgency ("Limited tickets")
✅ Test multiple variations
✅ Optimize for mobile first
✅ Add captions to videos
✅ Keep text minimal (< 20%)
```

### Don'ts:

```
❌ Use stock photos (use real band photos)
❌ Hide important info (date, venue, price)
❌ Make it too busy (one message per ad)
❌ Forget mobile preview (most views on mobile)
❌ Use low-res images (minimum 1080px)
❌ Ignore brand colors (stay on-brand)
❌ Skip A/B testing (always test)
❌ Forget CTA (tell them what to do)
```
